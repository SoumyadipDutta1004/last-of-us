## Install Using NPM

### Clone the Repository
```sh
git clone https://github.com/SoumyadipDutta1004/last-of-us.git
cd last-of-us
```

### Install Dependencies
```sh
npm install
```

### Run the Application
```sh
npm run dev
```

Now, open your browser and navigate to `http://localhost:5173` to use the todo list.
